# Alumni Management system 

## Core modules 
### User management
•  Alumni registration & login 
•  Admin panel 
•  Role-based access (Admin / Alumni / Guest)
### Alumni profile 
•  Personal & contact details 
•  Education history 
•  Current job & company 
•  Profile photo upload 
### Job Portal 
•  Alumni can post & apply for jobs 
•  Company & role listings
### Event Management
•  Create/manage reunions, seminars 
•  RSVP / registration for events
### News & Announcements
•  Admin posts updates/news 
•  Notifications to alumni
### Donation Management 
• Track contributions to the institution
## Tech Stack 
Java , spring boot , maven , mysql , Thunder Client (to check backend)

## Tech

Dillinger is built with:

- [Next.js 14](https://nextjs.org/) — React framework with App Router
- [React 18](https://react.dev/) — UI components
- [Monaco Editor](https://microsoft.github.io/monaco-editor/) — VS Code's editor
- [markdown-it](https://github.com/markdown-it/markdown-it) — Markdown parser with plugins
- [Zustand](https://zustand-demo.pmnd.rs/) — State management with localStorage persistence
- [Tailwind CSS](https://tailwindcss.com/) — Styling
- [TypeScript](https://www.typescriptlang.org/) — Type safety
- [Lucide React](https://lucide.dev/) — Icons

Deployed on [Vercel](https://vercel.com/).
